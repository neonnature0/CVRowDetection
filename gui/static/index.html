Tips for Best Results
1 Draw tight polygons — exclude buildings, roads, and non-vineyard areas from block boundaries
2 Use Blind annotation — creates unbiased ground truth, avoiding circular dependency with the ML model
3 Centre on canopy, not brightness — sun angle shifts the bright side off-center from the actual row
4 Annotate diverse blocks — a few challenging blocks teach more than ten easy ones
5 Tune on one block first — experiment with the Tune panel, then apply your best settings as defaults
6 Enable Ensemble Confidence — when ML and Gabor agree, detection is reliable; disagreement flags blocks for review
